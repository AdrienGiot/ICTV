What is ICTV ?

  • ICTV is a Content Management System for digital signage.
  • In other words, ICTV allows you to create and modify content that will be displayed on screen networks.
  • ICTV is web-based. Everything from the management of the content to its reproduction on screens is done in a web-browser.
  • ICTV is easily extendable, with a simple and powerful plugin system.

Main concepts

To understand how ICTV works, we have to explain its key concepts. We will look at three main components an user can interact with.

  • Plugins
  • Channels
  • Screens

Plugins

They are at the heart of ICTV. Plugins are programs that bring content to the system.

  • Plugins can be automated programs, e.g. a program creating content based on an RSS feed.
  • But they can also interact with users and embed their own web-application inside ICTV.
  • All plugins can be configured with ease within the ICTV interface.

Plugins

ICTV already comes with useful plugins.

  • editor is a fully featured web editor to create slideshows.
  • embed allows any web page to be embedded into the system.
  • rss is an RSS parser creating content based on an RSS feed.
  • cal creates announcements based on Google Calendar events.
  • img-grabber easily extracts images from web pages and reproduces them in a stylish way.

Channels

They are used to organize content in the system. Much like TV channels, they allow various themed information to be brought in and distributed over the screen networks.

  • Each channel is an instance of a particular plugin.
    • A channel is a customization of a plugin.
  • Channels can be public or private to certain users.

Screens

A screen is an output displaying content. It may be physical (e.g. a TV) or purely virtual (e.g. a web page).

Content is brought to screens via subscriptions. When subscribed to a channel, a screen will display all its content.

Subscribing is thus an easy way to assign themed content to particular screens.

User management

All the responsibilities involved in managing channels, plugins and screens can be delegated to specific users at a fine level. In fact this very same responsibility can also be delegated too.

Accounts can be deactivated separately.
